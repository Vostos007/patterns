    preserve_sentences=True,  # –ù–µ —Ä–∞–∑—Ä—ã–≤–∞—Ç—å –ø—Ä–µ–¥–ª–æ–∂–µ–Ω–∏—è
    preserve_paragraphs=False # –ú–æ–∂–Ω–æ —Ä–∞–∑—Ä—ã–≤–∞—Ç—å –∞–±–∑–∞—Ü—ã

### –°—Ç—Ä–∞—Ç–µ–≥–∏–∏ chunking

#### 1. SEMANTIC (—Ä–µ–∫–æ–º–µ–Ω–¥—É–µ—Ç—Å—è!)
