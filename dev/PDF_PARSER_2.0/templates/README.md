
KPS — Knitting Pattern Standard (RU/EN/FR)
=========================================

В архиве два подхода:
1) latex/ — полностью открытая сборка на XeLaTeX с единым шаблоном.
2) indesign/ — артефакты для InDesign+CAT: Style Spec, GREP-правила, два JSX-скрипта.
3) xliff/ — пример XLIFF-файлов для интеграции с CAT.

Рекомендуемый рабочий процесс описан в вашем ответе в чате.
