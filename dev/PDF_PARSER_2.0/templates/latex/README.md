
# KPS (Knitting Pattern Standard) — XeLaTeX Stack

Сборка:
1) Установите XeLaTeX и шрифты `Noto Serif` и `Noto Sans` (или замените в preamble.tex на установленные у вас).
2) В каталоге `latex/` запустите: `make ru` (или `make en`, `make fr`).
3) Полученный PDF будет собран с одинаковой вёрсткой и стилями для любого языка.

Принципы:
- Только привязанные к абзацам рисунки (см. `\KPSFigure`).
- Единые поля, кегли и стили заголовков (см. `preamble.tex`).
- Таблицы размеров и материалов оформляются одними и теми же макросами.

Глоссарий — `glossary.csv` (RU/EN/FR).
